## 🚀 Deployment Instructions

1. **Install/Build**: `pnpm install && pnpm build`
2. **Test Locally**: `pnpm dev` (http://localhost:3000)
3. **Check Meta**: Inspect page source for schema.org tags
4. **Verify Sitemap**: Navigate to `/sitemap.xml`
5. **Check robots.txt**: Navigate to `/robots.txt`
6. **Deploy**: `pnpm deploy` or GitHub integration

3. Monitor search console for crawl errors
